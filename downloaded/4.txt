     <h1>Bugs</h1> 
     <p>You can review existing issues (including bugs and new features) or file a new bug via the <a href="https://github.com/jhy/jsoup/issues?q=sort%3Acreated-desc"><b>issue list</b></a>.</p> 
     <p>If you are not sure if something is a bug, or need a hand, please <a href="discussion"><b>join the discussion</b></a>.</p> 
     <p>Please do not create a GitHub issue for questions on how generally to use jsoup; those issues will be closed.</p> 
     <h3>What constitutes a bug?</h3> 
     <p>The goal of jsoup is that is will sensibly parse any real-world HTML page into effectively the same parse tree that a modern browser would.</p> 
     <p>If you find a page that jsoup doesn't parse correctly, or if it doesn't behave as the <a href="/apidocs/">documentation</a> says it will, then that's a bug, and your assistance by way of filing it is welcomed.</p> 
     <p>For the fastest fix, please supply your code and the input HTML that triggers the issue.</p>
