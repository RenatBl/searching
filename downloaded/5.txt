     <h1>FAQS and getting help</h1> 
     <p>There are many helpful answers to <a href="https://stackoverflow.com/questions/tagged/jsoup?tab=Votes"><b>jsoup FAQs</b></a> on Stack Overflow.</p> 
     <p>If you have a question on how to use <code>jsoup</code>, please post it on <a href="https://stackoverflow.com/questions/tagged/jsoup?tab=Votes"><b>Stack Overflow</b></a>. Use the <code>jsoup</code> tag.</p> 
     <p>Please do not create a GitHub issue for questions on how generally to use jsoup; those issues will be closed.</p> 
     <h1>Discussing new ideas and changes</h1> 
     <p>If you have a new feature idea for jsoup, or would like to discuss a change proposal, or have other feedback, use <a href="https://github.com/jhy/jsoup/issues?q=label%3Adiscussion">GitHub issues</a> with the <code>discussion</code> label. Discussion is welcomed!</p>
