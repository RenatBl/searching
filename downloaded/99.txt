      <div class="block">
        The attributes of an Element. 
       <p> Attributes are treated as a map: there can be only one value associated with an attribute key/name. </p> 
       <p> Attribute name and value comparisons are generally <b>case sensitive</b>. By default for HTML, attribute names are normalized to lower-case on parsing. That means you should use lower-case strings when referring to attributes by name. </p>
